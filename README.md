# GreenTech MVP Sostenible

## Qué he hecho
He refactorizado el index.html eliminando dependencias innecesarias como Bootstrap, jQuery y Moment.js, sustituyéndolas por HTML, CSS y JavaScript nativo.

## Mejoras aplicadas
- Eliminado Bootstrap 
- Eliminado jQuery + Moment 
- Eliminado Font Awesome 



- JS reducido a nativo
- El .hero esta optimizado, antes era gigante
- sustituimos utilidades de bootstrap con un flexbox
- Creamos un botón con CSS implementado en el index.html
- Optimizamos el footer
- Con un par de Scripts de JS insertamos el año automaticamente en el footer sin necesidad de usar JQuery o 	Moment.js



Esto contribuye a:
- Menor huella de carbono (menor rendimiento = menor energia)
- Mayor vida útil del hardware (no se sobrecalienta el dispositivo y la bateria dura mas)

## Relación con economía circular
Un software más ligero:
- Reduce la necesidad de hardware potente
- Evita la obsolescencia programada
- Disminuye el consumo energético global
