# GreenTech MVP Sostenible

## Qué he hecho
He refactorizado una landing page eliminando dependencias innecesarias como Bootstrap, jQuery y Moment.js, sustituyéndolas por HTML, CSS y JavaScript nativo.

## Mejoras aplicadas
- Eliminación de frameworks pesados
- Reducción del tamaño de imágenes
- Uso de CSS nativo
- Sustitución de librerías JS por código propio
- Reducción del consumo de datos

## Impacto ambiental
La reducción del peso de la web disminuye el consumo energético en:
- Dispositivos de usuario
- Redes de transmisión
- Centros de datos

Esto contribuye a:
- Menor huella de carbono
- Mayor vida útil del hardware
- Menos residuos electrónicos

## Relación con economía circular
Un software más ligero:
- Reduce la necesidad de hardware potente
- Evita la obsolescencia programada
- Disminuye el consumo energético global
